# Indoor-Navigation-System using Unity-
This project intends to solve this by designing and building an indoor Navigation system, which we achieve with the assistance of a mobile device's camera and gyroscope, our approach would be to use AR Core to show the navigation path to the user on his mobile phone along with a 2D mini-map for guidance where we track the initial position of the user by making user scan a QR code at a specific start location. And another objective of this project is not to depend on any specific hardware or Internet access for navigation. 


To Run :-

Clone and open in Android Studio.
